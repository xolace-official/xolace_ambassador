# Xolace Ambassadors Program

The Ambassadors website for Xolace Inc . Everything about the Ambassadors program will be on this website

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
